Javascript


Notions
  • Affiche une information dans le devTools, onglet console. 
     console.log();
  • Stop l'exécution du script : 
    debugger;
  • Afficher un message dans une boite de dialogue : 
    alert('message');
  • Entrer un message dans une boite de dialogue : 
    cont Ma_variable = prompt('entrer un message');
  • Récupérer le message : 
    alert(`Bonjour, ${prenom}`);
    ou 
    alert(Ma_variable);
  • Type nombre (number) : valeur numérique entière ou réelle
  • Type chaîne de caractères (string)
  • Une valeur de type texte est délimitée par une paire de guillemets simples ' ... ' ou doubles " ... ".
  • Les opérateurs + , - , * et / permettent de réaliser des opérations arithmétiques entre deux nombres. Appliqué à deux chaînes, l'opérateur + les fusionne en une seule. Cette opération est appelée concaténation.
  • Les commentaires ( // ... ou /* ... */ )

les variables

  • Choisir des noms significatifs
  • Bannir les caractères accentués
  • Ne pas utiliser les noms réservés du langage
  • Adopter une convention de nommage

3 propriétés:

  • nom : peut contenir des lettres majuscules ou minuscules, des chiffres (sauf en première position) et certains caractères comme le dollar ($) ou le tiret bas, appelé underscore (_).
  • valeur : la donnée actuellement mémorisée dans cette variable.
  • type : détermine le rôle et les opérations applicables à cette variable.

Déclaration

  • var ma_variable (obsolète)
  • let ma_variable
  • const ma_variable (pour une valeur fixe)

Affectation, assignation de valeur

  • Assigne une valeur ma_variable 
    ma_variable = value
  • Ajoute value à ma_variable 
    ma_variable += value
  • Ajoute 1 à ma_variable 
    ma_variable++
