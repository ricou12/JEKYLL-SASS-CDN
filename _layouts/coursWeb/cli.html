CLI

Command line interface


Définition

CLI est un programme en ligne de commande ou une interface homme-machine dans laquelle la communication entre l'utilisateur et l'ordinateur s'effectue en mode texte.

Notions

Chemin relatif : par défaut à partir du répertoire racine cependant on peut remonter plus haut dans l'arborescence. 

ex: ./../root

Chemin absolu : à partir de la racine du disque.

Les commandes

  • pwd : renvoie le répertoire courant.
  • ls : liste les fichiers et dossiers du répertoire courant
    • ls -l : affiche sous forme de liste

      • $ ls -la
total 4
drwxr-xr-x 1 Eric Lanza 197121 0 oct. 29 20:25 ./          : répertoire courant
drwxr-xr-x 1 Eric Lanza 197121 0 oct. 29 20:25 ../         : répertoire parent
-rw-r--r-- 1 Eric Lanza 197121 6372 oct. 29 22:11 404.html : "-" pour fichier
drwxr-xr-x 1 Eric Lanza 197121 0 oct. 29 21:00 assets/     : "d" pour dossier

    • ls -a : affiche les fichiers et dossiers cachés
    • ls -la : combine les 2 options précédentes
  • cd : se déplacer dans l'arborescence des dossiers
    • cd < nom_répertoire > : ouvre le dossier concerné
    • cd .. : remonte dans l'arborescence
    • cd ~ : retourne au répertoire racine (root)
  • touch < nom_fichier > : crée un fichier
  • mkdir < nom_repertoire > : crée un dossier
  • cat < nom_fichier > : affiche le contenu d'un fichier
  • less, more : affiche aussi le contenu d'un fichier
  • mv < source > < destination > : déplace un dossier, fichier
  • cp < source > < destination > : copie un fichier ou dossier
  • rm < nom_fichier >
  • rm -rf < nom_fichier >
  • grep < recherche > <nom_fichier > : trouve un mot dans un fichier
  • man : manuel navigation fleche haut et bas q pour quitter

Raccourcis clavier

  • Flèche haut : navigation dans l'historique des commandes
  • tab : autocomplétion
  • ctrl + r : recherche dans l'historique
