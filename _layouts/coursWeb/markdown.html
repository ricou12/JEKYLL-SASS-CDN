Les listes

    Liste simple :
    * Élément 1
    * Élément 3
    * Sous-élément 1
    * Sous élément 2
    * Élément 2
    
    Liste ordonnée :
    1. Élément 1
    2. Élément 2
    1. Sous-élément 1
    2. Sous élément 2
    3. Élément 3
    
    Listes simples et ordonnées imbriquées :
    1. Élément 1

Checklist / Liste de cases à cocher 

    - [x] Élément 1, coché
    - [ ] Élément 2, non-coché

    - [ ] Sous-élément 2.2, non-coché
    - [ ] Élément 3, non-coché

Échapper les caractères spéciaux

    \\ : Anti-slash
    \` : Apostrophe curve
    \* : Asterisk
    \_ : Low dash / Underscore
    \{\} : Braces
    \[\] : Hooks
    \(\) : Parentheses
    \# : Sharp
    \+ : Sign more
    \- : Less sign / hyphen
    \! : Exclamation dot
    \. : Dot
